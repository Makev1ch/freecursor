## Features

- Customizable keybind (F6 by default)
- Return via LMB or ESC
- HUD hiding (Simulate F1)
- Background blur removal
- English and Russian localization

## Usage

1. Press **F6** (or your custom key) to release the cursor.
2. Press **LMB** or **ESC** to return.

## Configuration

### Keybind:
`Options > Controls > Key Binds > FreeCursor`

### Settings:
- **In-game:** Press **N** to open built-in config.
- **ModMenu:** Configure via ModMenu interface.
